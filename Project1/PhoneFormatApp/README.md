## Project 1 (CS-478) — Phone Format Checker (Java)

### What this is
Android app with **2 activities**:
- `MainActivity`: welcome text + 2 buttons (portrait: vertical buttons, landscape: horizontal buttons).
- `PhoneEntryActivity`: user enters a phone number; validates the required formats and returns a result to `MainActivity`.

### How to run (Android Studio)
- Open the folder `Project1/PhoneFormatApp` in Android Studio.
- Use a **Pixel 9 AVD, Android 16 (API 36)**.
- Sync Gradle, then Run.
